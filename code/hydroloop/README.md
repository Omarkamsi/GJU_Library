# HydroLoop

Public water + energy intelligence for German Jordanian University.
IEEE Region 8 Sustainable Universities Program 2026.

## Quickstart

```bash
cp .env.example .env
docker compose up
# API:  http://localhost:8000
# Web:  http://localhost:3000
# MQTT: localhost:1883
```

See `docs/architecture.md` for the full design.
