Our Solution — Two Layers, One Public Dashboard

Sense it. Stream it. Show it.

Layer 1 — Wheel Observatory

A non-invasive sensor unit beside the central water wheel.

  • PZEM-004T clamp · motor power
  • JSN-SR04T ultrasonic · water level
  • BME280 · weather model
  • ESP32-CAM · presence flag

Layer 2 — Building Flow Network

Six Hall-effect flow meters at building water inlets.

  • Engineering, Library, CS
  • Architecture, Business, Medical
  • Per-building L/min · running totals
  • Anomaly & leak detection

Public Dashboard

Live cloud platform anyone can visit.

  • FastAPI + TimescaleDB
  • MQTT ingest · 10-second cadence
  • Server-Sent Events · live charts
  • QR code on campus signage

Live · streaming right now

Why HydroLoop wins

Measurable. Replicable. Public.

01Measurable impact

Every kWh and every liter is logged. The IEEE final report is generated from the database — not estimated, not assumed.

02Replicable across the region

Open-source software, off-the-shelf $80 hardware nodes, and a written bench guide mean any university in Region 8 can clone it in a semester.

03Public by design

The dashboard is a public URL. A QR plaque next to the wheel lets every passing student see what HydroLoop is showing — sustainability stops being abstract.

04On budget, on time

$1,400 in hardware. 12 weeks of execution. Software stack already shipped & running in week 1 — sensors install on top of it as they arrive.
